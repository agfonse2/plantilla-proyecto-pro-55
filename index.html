Acerca de nosotros


Mediante el uso de nuestro centro de tutorías, puedes aprender y ejecutar código informático experimentalmente, para ver qué funciona y qué no, antes de implementarlo. Los diagramas paso a paso son probablemente la forma más popular y fácil de seguir para mostrar cómo doblar las cosas fuera de papel. Para ayudar a su búsqueda hemos reunido la mayor colección de diferentes lenguajes técnicos en Internet.

HTML

  1. HTML es el lenguaje estándar marcado para crear páginas Web.
  2. HTML describe la estructura de una página web.
  3. HTML consiste en una serie de elementos.

CSS

  • CSS es el lenguaje que usamos para dar estilo a un documento HTML.
  • CSS describe como los elementos HTML deben ser mostrados.
  • CSS ahorra mucho trabajo. Puede controlar el diseño de varias páginas web a la vez.

Javascript

  1. JavaScript es el lenguaje para programar en la web.
  2. JavaScript es fácil de aprender.

PHP

  • PHP 7 es la última versión estable lanzada.
  • Es una poderosa herramienta para hacer páginas Web dinámicas e interactivas
  • PHP es una alternativa libre y eficiente, ampliamente utilizada a competidores como el ASP de Microsoft.

Python

  1. Python se puede utilizar en un servidor para crear aplicaciones web.
  2. Python se puede utilizar junto con el software para crear flujos de trabajo.
  3. Python tiene una sintáxis simple similar a la del inglés.

React Native

  • React Native soporta las plataformas IOS y Android.
  • Con React Native, no construyes una aplicación web móvil, una aplicación HTML5, o una aplicación híbrida.

Bootstrap

  1. Bootstrap es el marco HTML, CSS y JavaScript más popular para el desarrollo de sitios web móviles de primera respuesta.
  2. ¡Bootstrap es completamente libre para descargarse y usarse!

p5.Js

  1. p5.js es una biblioteca JavaScript para la codificación creativa, e inclusiva para artistas, diseñadores, principiantes, y cualquier otro!
  2. p5.js es libre y de código abierto porque creemos que el software, y las herramientas para aprenderlo, deben ser accesibles para todos.
